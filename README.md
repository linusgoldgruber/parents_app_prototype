# ParentsAPP

ParentsAPP is a prototype for an inclusive, AI-supported family decision-making platform.

The project is focused especially on co-parenting and multi-household family coordination. The main goal is to help parents, children, grandparents, and other caretakers communicate more clearly, coordinate shared responsibilities, and feel included in family decisions.

## Core Prototype Scope

- role-based family participation
- shared communication
- shared calendar and events
- decision support through voting
- document sharing with visibility controls
- AI-supported tone assistance and mediation
